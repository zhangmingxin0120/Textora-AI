# Textora 智文 ⭐

> AI生成内容一键转Word，解决公式乱码、代码格式、表格排版问题

## 🌟 功能特点

- ✅ **完美支持 DeepSeek、豆包、ChatGPT** 等主流 AI 平台
- ✅ **公式不乱码**：自动将 LaTeX 公式转换为 Word 原生公式
- ✅ **代码高亮保留**：代码块格式完整，语法高亮清晰
- ✅ **表格排版完美**：自动优化表格样式
- ✅ **隐私安全**：转换过程不上传服务器，本地处理

## 🚀 使用方法

1. 访问 https://textora.cn
2. 上传 .md 文件或粘贴 Markdown 内容
3. 一键下载完美 Word 文档
